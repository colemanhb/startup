    About LibreBoox

    LibreBoox is inspired by a couple different things I've learned about learning languages. First of all, reading is one of the best ways to learn a language. Second, seeing definitions of words can be even better than seeing translations because it keeps you thinking in the target language. I used all free books from Project Gutenberg available in foreign languages, some of which are originally in those languages while others are translations.

    I made this website because it's the type of website I would use to learn languages. I always feel drawn to classic literature, and reading them in foreign languages is a great way to kill two birds with one stone, especially if the book was originally written in a language other than English.
